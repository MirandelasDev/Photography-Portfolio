# Photography-Portfolio
Portfólio profissional desenvolvido para fotógrafo, com foco em apresentação visual, navegação fluida e valorização do conteúdo visual. O projeto utiliza uma estrutura simples e eficiente para exibição de imagens e contato direto com clientes por meio de integração com FormSubmit.

# Portfólio para Fotógrafo

Projeto de um site de portfólio desenvolvido para um fotógrafo, com o objetivo de apresentar trabalhos de forma profissional, moderna e visualmente atrativa.

## Sobre o Projeto

Este projeto consiste em um site responsivo focado na exibição de fotos e vídeos, priorizando clareza visual, organização e facilidade de navegação.

O site funciona como uma vitrine digital, permitindo que o fotógrafo apresente seus trabalhos e receba contatos de forma simples, sem necessidade de backend.

## Funcionalidades

* Galeria de imagens
* Seção de projetos ou vídeos
* Layout responsivo
* Animações suaves durante a navegação
* Integração com redes sociais
* Formulário de contato funcional com envio via FormSubmit

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* FormSubmit (envio de formulário sem backend)
